# Travel-Management
Note: Single-file structure is used to meet project requirements; normally, the code would be split into multiple files.

## Description

Console application for managing travel records stored in a CSV file.

## Features

 add / edit / delete records
 sort by date
 find by price
calculate average price


## Commands

 add ID;City;Date;Days;Price;Vehicle
 edit ID;...
 del ID
 print
 sort
 find PRICE
 avg
 exit
